# Бот-ассистент (Gemini-1.5-flash) 🤖
>*UPD : Смотрите какие модели вам доступны по вашему API, у меня все встало на версии
-> GEMINI-1.5-FLASH*
Бот на aiogram 3, который общается с нейросетью Google Gemini. Память (контекст диалога) хранится в Redis.

## ⚠️ Нюанс: Прокси для Gemini
Google API из РФ работает с перебоями, поэтому боту нужен прокси. 
Чтобы всё завелось, создай файл `.env` (смотри пример в `.env.example`) и добавь туда строку с твоим прокси:
PROXY_URL=http://логин:пароль@айпи:порт
